# Документация проекта Tala

| Поле | Значение |
| --- | --- |
| Цель | Дать обзор структуры знаний по проекту и быстрые ссылки |
| Аудитория | Разработчики, AI агент, авторы контента |
| Ответственный | Команда разработки |

## Как пользоваться

- Используй разделы ниже, чтобы быстро перейти к нужной теме.
- В конце каждого документа смотри блок «Что менять при доработке» — он помогает учесть связанные компоненты при изменениях.
- Если добавляешь новый модуль или процесс, создай документ и добавь ссылку на него в соответствующий раздел.

## Добавление новой документации

1. Определи расположение: выбери существующий раздел или создай новый.
2. Добавь в начало таблицу с метаданными (цель, аудитория, дата).
3. Свяжи документ с кодом: перечисли ключевые классы и связанные файлы.
4. Обнови эту страницу, чтобы включить новый материал в навигацию.
